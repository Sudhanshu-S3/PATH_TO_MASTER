### Longest Increasing Subsequence

[problem link](https://leetcode.com/problems/longest-increasing-subsequence/description/)

Find the length of the longest strictly increasing subsequence.
A subsequence keeps the original order but can skip elements.

---

#### Approach 1 — O(n^2) DP

`dp[i]` = length of the longest increasing subsequence that **ends** at index `i`.

- Every element alone is a subsequence of length 1, so `dp[i]` starts at 1.
- For each `i`, look at all previous `j < i`. If `store_[j] < store_[i]`,
  we can extend that subsequence: `dp[i] = max(dp[i], dp[j] + 1)`.
- The answer is the largest value in `dp`.

```cpp
#include <iostream>
#include <vector>
#include <algorithm>

int solve(std::vector<int> &store_)
{
    int n = store_.size();
    std::vector<int> dp(n, 1);

    int best = 1;
    for (int i = 1; i < n; i++)
    {
        for (int j = 0; j < i; j++)
        {
            if (store_[j] < store_[i])
            {
                dp[i] = std::max(dp[i], dp[j] + 1);
            }
        }
        best = std::max(best, dp[i]);
    }

    return best;
}

int main()
{
    int num_;
    std::cin >> num_;

    std::vector<int> store_(num_, 0);

    for (auto &a_ : store_)
    {
        std::cin >> a_;
    }

    if (num_ == 0)
    {
        std::cout << 0 << "\n";
        return 0;
    }

    std::cout << solve(store_) << "\n";
    return 0;
}
```

#### Approach 2 — O(n log n) patience sorting

Keep a `tails` array where `tails[k]` is the smallest possible tail value of an
increasing subsequence of length `k + 1`.

- For each number, binary-search (`lower_bound`) for the first tail `>= x`.
- If found, replace it with `x` (a smaller tail is always better).
- If not found, `x` extends every subsequence, so append it.

`tails` is not the actual subsequence, but its **length** is the LIS length.

```cpp
#include <iostream>
#include <vector>
#include <algorithm>

int solve(std::vector<int> &store_)
{
    std::vector<int> tails;

    for (int x : store_)
    {
        auto it = std::lower_bound(tails.begin(), tails.end(), x);
        if (it == tails.end())
        {
            tails.push_back(x);   // x extends the longest chain
        }
        else
        {
            *it = x;              // shrink the tail for this length
        }
    }

    return tails.size();
}
```

> Use `upper_bound` instead of `lower_bound` if the subsequence may be
> non-decreasing (duplicates allowed).
