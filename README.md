# PATH_TO_MASTER

A space to keep a grip on the fundamentals in one place. Learning out loud.

Building this is how I organize my own understanding. Writing a topic down is
what shows me the gaps in it.

### What this covers

- C++ fundamentals
- Advanced C++
- Low-latency concepts and tools
- Data structures and algorithms

Notes include theory, implementations, and small projects where they help, along
with good questions and good resources worth keeping. Blogs, technical analysis,
and whitepapers get added as I go.

### How I work on it

- No fixed schedule. I move at my own pace, with minimal use of AI.
- Code here is for learning, so expect rough edges. If you find something
  interesting or spot an issue, lmk.
